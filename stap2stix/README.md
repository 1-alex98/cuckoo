# STAB2STIX
- use python2.7

```
usage: stap2stix.py [-h] --stap STAP [--output OUTPUT]

Parse system calls to observables.

optional arguments:
  -h, --help            show this help message and exit
  --stap STAP, -s STAP  path to strace output
  --output OUTPUT, -o OUTPUT
                        the file the results are written to
```
